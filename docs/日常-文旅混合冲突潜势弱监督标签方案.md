# 日常-文旅混合冲突潜势弱监督标签方案

我们这一步的目标不是提出一个成熟的“冲突指数”，而是先做一套**可解释的弱监督标签**，给后续 GNN 模型训练使用。

核心判断：

```text
日常-文旅混合冲突潜势
= 日常生活功能和文旅功能同时高
× 地铁客流或路网通行压力高
```

也就是说，只有“居民日常使用”和“游客/文旅消费使用”在同一街段附近叠加，并且这个地方又有较强人流或通行压力时，才认为它更可能产生混合冲突。

## 一、最终采用的公式

对每条街段 `i`，计算 4 个基础指标。

## 1. 日常生活强度 Daily

统计街段附近的日常 POI，例如便利店、菜场、超市、药店、学校、医院、社区服务、快递、维修、银行等。

```text
Daily_i = [Σ max(0, 1 - d_ip / 100)] / Length_i
```

解释：

- `d_ip` 是街段 `i` 到日常 POI `p` 的距离；
- 100m 内的 POI 会被计入；
- 越靠近街段，贡献越大；
- 超过 100m，贡献为 0；
- 最后除以街段长度，避免长街段天然 POI 更多。

## 2. 文旅吸引强度 Tourism

统计街段附近的文旅 POI 和历史文化节点。

文旅 POI 包括：

```text
咖啡、餐饮、酒吧、文创、展馆、美术馆、酒店等
```

历史文化节点包括：

```text
历史建筑、文物保护单位、风貌道路、名人故居等
```

公式：

```text
Tourism_i =
[Σ max(0, 1 - d_ip / 100)
+ 2 × Σ max(0, 1 - d_ih / 100)] / Length_i
```

解释：

- 普通文旅消费 POI 权重为 1；
- 历史文化节点权重为 2；
- 这样做是因为历史建筑、文保单位、风貌道路等比普通商业 POI 更能代表稳定的文旅吸引力。

## 3. 地铁客流压力 Metro

统计街段 500m 范围内地铁站客流带来的压力。

```text
Metro_i = Σ Flow_s × max(0, 1 - d_is / 500)
```

其中：

```text
Flow_s = 日均进站量 + 日均出站量
```

解释：

- 离高客流地铁站越近，压力越大；
- 500m 外认为影响为 0；
- 如果一个街段附近有多个地铁站，则把多个站点影响加总。

## 4. 路网通行压力 Network

用街段的 edge betweenness centrality 表示。

```text
Network_i = EdgeBetweenness_i
```

通俗解释：

如果一条街段经常出现在很多最短路径上，说明它更可能承担穿行、换乘、绕行等通行压力。

## 二、标准化

四个指标统一标准化到 0-1：

```text
D_i = Norm(Daily_i)
T_i = Norm(Tourism_i)
M_i = Norm(Metro_i)
N_i = Norm(Network_i)
```

标准化公式：

```text
X_norm = (X - X_min) / (X_max - X_min)
```

## 三、混合冲突潜势公式

先算“日常-文旅共存强度”：

```text
Mix_i = 2D_iT_i / (D_i + T_i + ε)
```

解释：

- 日常高、文旅也高，`Mix` 才高；
- 只有日常高但文旅低，不算高混合；
- 只有文旅高但日常低，也不算高混合。

再算“外部压力”：

```text
Pressure_i = 1 - (1 - M_i)(1 - N_i)
```

解释：

- 地铁压力高，Pressure 会高；
- 路网压力高，Pressure 也会高；
- 两者都高，Pressure 更高；
- 两者都低，Pressure 低。

最终冲突潜势：

```text
Conflict_i = Mix_i × Pressure_i
```

完整公式：

```text
Conflict_i =
[2D_iT_i / (D_i + T_i + ε)]
×
[1 - (1 - M_i)(1 - N_i)]
```

## 四、标签划分

按照 `Conflict_i` 的分位数直接划分：

```text
high：前 20%
medium：中间 40%
low：后 40%
```

也就是：

```text
Conflict_i >= 80% 分位数 → high
40% 分位数 <= Conflict_i < 80% 分位数 → medium
Conflict_i < 40% 分位数 → low
```

这样做的好处是类别数量比较稳定，适合后续作为 GNN 的弱监督标签。

## 五、方法依据

这个公式不是凭空造的，而是把几个已有研究思路组合起来。

### 1. POI 距离衰减 / 可达性思想

日常 POI、文旅 POI、地铁客流都采用“离得越近影响越大”的距离衰减逻辑。这个思路来自交通可达性和机会可达性研究。

来源：

- Hansen, W. G. (1959). *How Accessibility Shapes Land Use*. Journal of the American Institute of Planners.
- Geurs, K. T., & van Wee, B. (2004). *Accessibility evaluation of land-use and transport strategies: review and research directions*. Journal of Transport Geography.
- 参考：[Accessibility in transport](https://en.wikipedia.org/wiki/Accessibility_%28transport%29)

### 2. 路网中心性 / Betweenness Centrality

路网压力用 betweenness centrality，表示一条街段在多少最短路径中被经过。

来源：

- Freeman, L. C. (1977). *A Set of Measures of Centrality Based on Betweenness*. Sociometry.
- Brandes, U. (2001). *A Faster Algorithm for Betweenness Centrality*. Journal of Mathematical Sociology.
- 参考：[Betweenness centrality](https://en.wikipedia.org/wiki/Betweenness_centrality)
- 参考：[Brandes' algorithm](https://en.wikipedia.org/wiki/Brandes%27_algorithm)

### 3. 城市街道网络中心性研究

城市街道可被看作网络，街段中心性可以解释其潜在通行重要性。

来源：

- Porta, S., Crucitti, P., & Latora, V. *The Network Analysis of Urban Streets: A Primal Approach*.
- 参考：[The Network Analysis of Urban Streets: A Primal Approach](https://arxiv.org/abs/physics/0506009)

### 4. 文旅压力与居民冲突的理论背景

旅游过度集中会影响居民生活质量，产生游客与本地居民之间的空间使用冲突。

参考：

- [Overtourism](https://en.wikipedia.org/wiki/Overtourism)

## 六、一句话总结

```text
我们最终采用的不是简单加权总分，而是：

Conflict = 日常-文旅共存强度 × 地铁/路网外部压力

它的含义是：
居民日常功能和文旅吸引功能同时高，
并且叠加地铁客流或路网通行压力时，
该街段的日常-文旅混合冲突潜势更高。
```
